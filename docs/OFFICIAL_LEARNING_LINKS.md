# Official Learning Links

This page collects official Swift, SwiftUI, and Xcode references so you can study the repository and the platform docs side by side.

## Recommended Study Order

1. Swift language basics
2. Swift collections, structs, enums, and methods
3. Protocols and generics
4. Swift concurrency
5. SwiftUI state and list-driven UI
6. SwiftUI navigation, search, and previews
