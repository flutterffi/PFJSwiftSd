# Study System Roadmap

This page is the practical roadmap for growing PFJSwiftSd from a strong practice repository into a more complete curriculum.

## What The Repository Already Does Well

- small runnable examples
- a layered path from syntax to app work
- official documentation links
- a real feature track through `StudyDashboardApp`
- a growing test story for the dashboard core

## What To Improve Next

### Priority 1

- strengthen missing Swift language fundamentals
- strengthen SwiftUI data-flow and layout coverage
- keep every new topic connected to a runnable example

### Priority 2

- add one short README for `language/`, `concurrency/`, and `architecture/`
- add explicit learning goals to each lesson file
- connect official docs to each folder, not only the root docs page

### Priority 3

- add milestone tags that reflect curriculum stages
- add a 7-day and 14-day sprint plan
- add a simple progress checklist for self-study

## Suggested Milestone Meaning

- `v0.1.0` initial training skeleton
- `v0.2.0` reusable dashboard core and tests
- `v0.3.0` minimal SwiftUI app shell
- `v0.4.0` lesson detail navigation
- `v0.5.0` official-aligned curriculum expansion

## Good Rule For Future Additions

Every new topic should answer all three:

1. What official concept does this teach?
2. What local lesson file demonstrates it?
3. What project or app feature uses it in context?
