# ReducerStyleApp

ReducerStyleApp is the action/state-driven comparison track.

## Training Goal

Compare MVVM against a reducer-style architecture with explicit state and actions.

## What To Observe

- how explicit the state flow becomes
- whether feature changes feel safer
- how much ceremony appears compared with MVVM

## Current Status

The first scaffold now includes:

- state
- actions
- reducer logic
- store shell
- list view-state projection
- detail selection and detail view-state projection

This makes the reducer-style track a stronger comparison point against both MVC and MVVM.
