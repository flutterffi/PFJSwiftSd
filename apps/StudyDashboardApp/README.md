# StudyDashboardApp

StudyDashboardApp is a minimal macOS SwiftUI shell that reuses the dashboard core from `projects/01_study_dashboard_app`.

## Current UI Scope

- dashboard list rendering
- search
- track filtering
- bookmark toggling
- lesson detail navigation
- loading and empty states

## Build

```bash
swift build --product StudyDashboardApp
```

## Run

```bash
swift run StudyDashboardApp
```
